(for version 6.0.0 or above)

Updating of PHP DataGrid is a very straight forward process:

[top]

Step 1. Backup old installation.

We strongly recommend to back up your old installation, before you start updating to new version.


[top]

Step 2. Uncompressing downloaded file.

Uncompress the PHP DataGrid version 6.x.x script archive. The archive will create a directory called "datagrid_6_x_x"


[top]

Step 3. Uploading and replacing existing files.

Upload content of this folder (all files and directories it includes) to your DataGrid directory using FTP or another tool.


For example, you have PHP DataGrid installaed here: 


        public_html/your_site/datagrid

Peplace the content of this folder with the the content of the new version.


[top]

Step 4. Testing.

Test your DataGrid files to be sure, that new version of PHP DataGrid works as expected. All recent changes and improvements could be found here or in docs/History of Changes.txt of the script archive.


Congratulations, you now have PHP DataGrid v.6.x.x. Updated!
